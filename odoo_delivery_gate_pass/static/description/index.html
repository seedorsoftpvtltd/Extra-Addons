Delivery Gate Pass / Guest Pass in Odoo

Delivery Gate Pass Record Management in Odoo

This module allows you to manage Delivery Gate Pass & Details under Gate/Security User.


Main Features :
  • Inventory Users can click the Generate Gate Pass Button then create the gate pass form Stock Picking / Delivery Order Form.
  • Inventory Users can click the Print Gate Pass Button and print the gate pass report from Stock Picking Form.
