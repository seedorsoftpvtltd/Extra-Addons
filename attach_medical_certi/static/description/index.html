Attach Medical Certificate for Sick Leave.

This module allows to attach documents like medical certificate in Time Off Request.

Features

  • Employee can attach documents like medical certificate in Time Off Request.
  • Enterprise
  • Community

Configuration

  • Select Certificate Required in Time Off Type to allow employee to attach certificate in Time Off Request.

    • Time Off Request

  • Here, employee must have to attach medical certificate for Sick Time Off.


  • Manager can download certificate and based on it manager can Approve/Refuse the Time Off Request.
