Schedule Mail Send Later Odoo Apps

Schedule Emails To Send Later Odoo Apps helps users to schedule send mail on selected date and time also user can add recipients and that email automatically links mail with chatter. Once a user sets a scheduled email it will set as scheduled email activity on document chatter. Once selected time pass schedule email automatically sent to all followers and recipients then mark as sent on chatter.

Features

Send Mail on Schedule Date.

User can schedule a mail with attachments and send on selected date.

Link Schedule Mail with Chatter.

Schedule mail automatically link with chatter, No need to send message manually.

Added Recipients

User can added recipients for sending mail.

Send Mail to All Followers and Recipients

User can send mail to added all followers and recipients.

Schedule Send Message Button

Click on schedule send message button then open wizard in that select schedule date time, write a subject and message also you can add recipients for sending mail.



Scheduled Mail

User can see scheduled mail under planned activities and sent to followers and recipients on scheduled date and time.

Sent Scheduled Mail

User can see sent scheduled mail to followers/recipients on the scheduled date, email will move to message chatter from planned activity after sending.

Added Recipients

User can see added recipients and followers shown in recipients field.



Sent Email to All Followers and Recipients

User can see sent email to all added followers and recipients.

Schedule Send Message with Attachments

Click on schedule send message button then open wizard in that select schedule date time, write a subject and message with attachment also you can add recipients for sending mail.



Scheduled Mail with Attachments

User can see scheduled mail under planned activities and sent to followers and recipients on scheduled date and time, they can also download attachment.

Sent Scheduled Mail

User can see sent scheduled mail to followers/recipients on the scheduled date, email will move to message chatter from planned activity after sending.

Added Recipients

User can see added recipients and followers shown in recipients field.



Sent Email to All Followers and Recipients

User can see sent email with attachments to all added followers and recipients.
