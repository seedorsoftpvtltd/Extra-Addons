Checklist for all stock Transfers

Checklist Functionality In All Transfers

Under Inventory --> configuration user will find two menuitems,
1. Inventory --> Configuration --> Checklist Template
2. Inventory --> Configuration --> Checklist Points

In Checklist Points, You can create the Pre-Defined Checklist points with clear Instruction

Tree view

In Checklist Template, user can create the Pre-Defined Templates and user can add the checkpoints inside the templates

Create the template and fill the name and code informations after that please click add a line

After clicking the add a line, it will popup all the checklist points, user can add the checklists according to the template

Tree view

In Transfer form user can select the template in checklist tab
After selecting template user can click the import checklist from checklist templete button

After clicking the button, it will bring all the checklist points related to the selected template

User can start working on checklist by click tick mark, user can make the checklist completed or Failed, if it is completed it will go to completed state in green colour, if it is Failed then it will go to failed state in red colour, in both state user can reset and he can work again in same checklist. also user can add the checklist manualy by clicking add a line.

User can print the checklist
