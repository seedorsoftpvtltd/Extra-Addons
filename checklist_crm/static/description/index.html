CRM Checklist

Checklist Points

In CRM configuration user will find two menuitems,
2. CRM --> Configurations --> Checklist Template
1. CRM --> Configurations --> Checklist Points

In Checklist Points, You can create the Pre-Defined Checklist points with clear instructions

Tree view

In Checklist Template, user can create the Pre-Defined Templates and user can add the checkpoints inside the templates

User can see the checklist tab in lead and opportunity form, there user can select the pre-defined checklist template, after selecting the template user need to click the button called Import Checklist From Checklist Template

After clicking the button, it bring all the checklist points related to checklist template
User can start working on checklist by click tick mark, user can make the checklist pass or fail, if it is pass it will go to completed state in green colour, if it is fail then it will go to failed state in red colour, in both state user can reset and he can work again in that checklist. he can also add the checklist points which is not in the template.

User can print the checklist
