Global Margin Percentage

Module Info

This module allows you to apply percentage wise margin for quotations and sales orders. This module adds a percentage margin from the products cost price. When you have a lot of items / products in your quote it will be difficult to adjust the price for each item, so this module will help you to apply a margin percentage to all the items in the quotation. The module also provide you the ability to modify the margin line item wise. So you can apply a common margin to all the items and later can change per line item as well. This module will be really helpful for your sales people when creating quotations as they can try different margin options before finalizing and sending the quote to customers.
