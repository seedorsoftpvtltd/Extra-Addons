Inspection for Job Order / Work Order

Construction and Job Contracting Projects

This app allow you to have inspection feature on your Job Orders for Construction and Job Contracting Projects.


Main Features:

  • Allow you to create job inspection for job order for your project.
  • You can setup inspection result and value as configuration.
  • Print job inspection report in pdf format.
  • For more details please watch Video & below screenshots.

Menus Available:

  • Jobs
    • Inspections
      • Job Inspections
    • Configuration
      • Inspection Values
      • Inspection Results
      • Inspection Types
