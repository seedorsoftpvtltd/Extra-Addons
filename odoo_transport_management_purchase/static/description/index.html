Odoo Transport Management for Purchase Orders and Incoming Shipments

Transport Management and Routes

This app allow you to manage transporter and transport details as per below features and screenshots.

Main Features:

  • System allow you to create Transporter(s) (res.partner model) using Transporters menu.
  • Allow you to configure Transport Location which will be used in Transport Routes setup.
  • You can set Transporter on Vehicle form so it will indicate Fleet/Vehicle belongs to specific Transporter.
  • Allow you to select Transporter (res.partner) on Purchase Quote / Order form which will be passed to Incoming Shipments / Pickings related to Purchase Order.
  • Allow you to select Vehicle, Parcels, Route, LR number, etc... on Incoming Shipments form / Picking.
  • When Incoming Shipments Validated by Warehouse User then system will create automatically Transport Entry.
  • Incoming Shipments / Picking PDF Report (Odoo out of box report) show Tranport details information.
  • Allow you to Reschedle anytime on your Transport Entry.
  • Allow you to halt anytime on your transport entry.
  • Reschedle will create new transport entry for related Picking/Incoming Shipments.
  • Transport Entry List / Form View with PDF report.
  • For more details please see video and below screenshots. For Sales Transport only you can check https://apps.odoo.com/apps/modules/12.0/odoo_transport_management/
