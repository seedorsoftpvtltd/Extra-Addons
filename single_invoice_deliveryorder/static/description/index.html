Single Invoice From Multi Delivery Order

The purpose of this module is to easily create single Invoice from the Multi Delivery Orders. You don't need to create Invoice manually.Just select Multi Delivery of same contacts and click on action for "Create Single Invoice" and you are ready to go.

Below listed are features of Single Invoice From Multi Delivery Orders module:

Prerequisites before Installation

There are no prerequisites required before the Installation

Configurations
