Odoo Employees Payslip Payment App

Odoo Payment from Payslip apps helps to create accounting Payment from the payroll Payslip for all the users and users have the choice to select payment journal, you can also create full or partial payments from the payslip.

Description

Added new Buttons and state in Payslip View

After adding the Payslip Details and by clicking on the Employee Payslip Payment button it will open the wizard.


Validation Error

If the user doesn't add Amount to Pay it will popup the validation.


Validation Error

If the user adds more than the Due Amount it will give a Validation Error.


Partial Payments of Due Amount

Users can do Partial Payment of Due Amount and By clicking on the Payslip Payments Smart button users can see the Journal of Partial Payments with particular Payslip.


Here, the User can also do full Payments of the rest of Payslips

Users can do the rest of the Payments by using Bank or Cash or any other Entries.


State Changed if Payments Completed

If the Payments gets completed of any Payslip that Payslip State will move in Paid state.



User Can see the Payslip Journal

By clicking on the Payslip Payments Smart button User can see the Payslip Journal Entries with the Particular Payslip.
