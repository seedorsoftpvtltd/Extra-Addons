Attendance Geolocation - Attendance Location

This module helps you to capture the employees attendance check in / check out Geolocation Coordinates information and generate a direct links to Google Maps from Employee Attendance.

Installation

- Copy attendance_geolocation_log module to addons folder

- Install the module normally like other modules

Features

  • Capture the Employees Attendance Geolocation on Check in / Check out
  • Geolocation link or Tree View and Form View if the Coordinates are available.
  • Capture the Employees Attendance Geolocation on Kiosk Mode.

1. User Configuration

User Menu -> Attendance Geolocation





2. Check In / Check Out

Attendances -> Tree View



Attendances -> Form View



Attendances -> Geolocation Link Open in new Browser.
