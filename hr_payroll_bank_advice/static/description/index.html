Payroll Bank Advice
Payment Bank Advice (HR Payroll) Odoo Module
(Only for Odoo Enterprise Edition )

This module add feature to give bank advice creation and send to bank. This module allow Hr to create bank advice directly from payslip batch.


Main Features:

  • This module add feature to give bank advice creation. This module allow Hr to create bank advice directly from payslip batch.

  • Advice record is created then user can send the email in a mail attached advice record.

  • User can print the Advice report from Advice record.
