Odoo Transport Management

Transport Management and Delivery Routes

This app allow you to manage transporter and transport details as per below features and screenshots.

Main Features:

  • System allow you to create Transporter(s) (res.partner model) using Transporters menu.
  • Allow you to configure Transport Location which will be used in Transport Routes setup.
  • You can set Transporter on Vehicle form so it will indicate Fleet/Vehicle belongs to specific Transporter.
  • Allow you to select Transporter (res.partner) on Sales Quote / Order form which will be passed to Delivery Order / Pickings related to Sales Order.
  • Allow you to select Vehicle, Parcels, Route, LR number, etc... on Delivery Order form / Picking.
  • When Delivery Order Validated by Warehouse User then system will create automatically Transport Entry.
  • Delivery order / Picking PDF Report (Odoo out of box report) show Tranport details information.
  • Allow you to Reschedle anytime on your Transport Entry.
  • Allow you to halt anytime on your transport entry.
  • Reschedle will create new transport entry for related Picking/Delivery Order.
  • Transport Entry List / Form View with PDF report.
