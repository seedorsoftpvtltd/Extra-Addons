Odoo Backend Debranding

This module is the base debranding module that allows you to change the look and feel of Odoo. You can configure things like company logo, favicon, page title, etc. to give it a new look and feel.

Odoo Backend Debranding

Salient Features
Configurable Backend Favicon and Title Brand

User can replace Odoo's default favicon and title from the backend with new ones of his choice.


Replace Odoo's logo From Backend

Replaces Odoo's logo from backend to your company's logo.


User Dropdown Menu Updated

Removes "My Odoo.com Account" option from the user drop down at backend.


Login Page Updated

Removes 'Odoo' from Backend Web login layout.


Renames "OdooBot"

Renames "OdooBot" to "System"


Backend Settings Page Updated

Removes 'Odoo' form Settings i.e Apps store icon, Social share, Implementation.


Backend Dialog Box View Updated

Replaces 'Odoo' from Dialog Box.


Odoo Planners Updated

Removes 'Odoo' from helps and planners.


Backend Odoo Notification Updated

Replaces 'Odoo' from odoo notifications.
