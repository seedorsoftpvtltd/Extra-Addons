While filing GSTR3B, either you need to fill the form manually or upload the JSON of details directly.

Time is precious so you need to save it; so file your GSTR3B in quick and easy steps. Hence, uploading JSON to file GSTR3B can help you to make the process quick and easy.

Odoo GSTR3B - Returns And Invoices have come up to save your time as well as efforts. You can simply select the time period and fetch invoices to generate GSTR3B Reports in JSON format.
