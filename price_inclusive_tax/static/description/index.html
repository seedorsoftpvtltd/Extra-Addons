Fix major issue for inclusive tax calculation(Group Type Tax) in Odoo.

As per Odoo default calculation for price inclusive tax,

Formula = base_amount - (base_amount / (1 + rate / 100))

So in case of group type taxes,


GroupTax = {base_amount - (base_amount / (1 + tax-1 / 100))} + {base_amount - (base_amount / (1 + tax-2 / 100))}

Which makes a wrong calculation.

Example :

  • base_amount = 100
  • rate(GroupTax) = 18% (9% + 9%)
  • Expected Tax : 15.2542373

    • As per default flow:
  • GroupTax = {100-(100/(1 + 9/100))} + {100-(100/(1 + 9/100))}
  • GroupTax = 8.25688073 + 8.25688073
  • GroupTax = 16.5137615

It makes 1.2595242 difference from expected amount.

So this module helps to calculate correct inclusive tax in case of group type taxes.
