Construction Daily Logs

Construction Daily Logs for Construction Business - Odoo App

Allow your Project user(s) and Project Manager(s) to create Daily logs for construction projects and job orders.

Main Features:

  • User(s) can Create and View logs from My Daily Logs Menu.
  • Manager(s) can Create and View logs from My Daily Logs and All Daily Logs Menu.
  • Mail with daily log report will be sent to all the followers by Send Report to Manager button.
  • User(s) can print a pdf report of 'Construction Daily Log' from the print Construction Daily Log menu.
  • For more details please see below screenshots and watch a video.

Menu Items Available:

  • Jobs
    • Daily Logs
      • My Daily Logs
      • All Daily Logs
