Work Package for Construction and Contracting

Work Package for Projects and Job Orders in Odoo

This app allows you to create and send work packages to your customers for your construction and contracting projects.


Main Features:

  • Work Package creation by adding multiple job orders / work orders to include in it.
  • Allow you to print a Work Package PDF report.
  • Allow you to Work Package send by email to your customer.
  • Work Package linked with Project and Job orders.
  • For more details please check below screenshots and video.

Menus Available:

  • Jobs
    • Job Orders
      • Work Packages
