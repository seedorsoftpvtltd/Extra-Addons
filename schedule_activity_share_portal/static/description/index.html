Portal Share for Schedule Activities

Schedule Activity Share to External Agents / Portal Users

My Account Portal Share Scheduled Activities to External Portal User / Agent

This app allows your external agents / portal user / contacts to view activities on my account portal of your website and send a message using portal login as shown in below screenshots.


Main features:

  • Sharing on Portal tab options added on Activity Types as a configuration.
  • Allow your internal users to share activities with external user / external portal users. These will help them to work on scheduled activities together with external agents.
  • Allow your external agents / portal users to view scheduled activities which are shared to them by internal users in the backend.
  • Allow portal agent / portal agent to send a message from the send message button.
  • Sent messages by portal users will appear in the portal messages smart button on activity form.
  • For more details please check below screenshots and watch the video.
