Change Order in Job Contracting and Construction

Change Order Management in Odoo Construction Projects and Job Orders

This module allows you to manage Change Order in Job Costing and Job Contracting and Construction Projects.

Main Features:

  • Allow Project/Site/Sales Users to create Change Order requests.
  • Approval Workflow : Draft->Confirm->Approve->Customer Approve->Done->Create Quotation.
  • Change Order can be Approved by Project Manager.
  • Allow you to print Change order in PDF and Send to your customer for Signature using open chatter by email.
  • Allow you to create sales quotations directly from the Change order form.
  • Fully integrated with the Sales App of Odoo.
  • Change order smart button on Project, Job Orders and Customer form.
  • For more details about job costing you can see the dependent app.
  • For more details please see below screenshots and watch video.
