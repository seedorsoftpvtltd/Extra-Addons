Employee User Auto Creation

This module provide Configurable Employee user auto creation

Configurable User autocreation for employee

Access Rights can be manage from job position itself.

Unlock more feature for login credentials by enabling user auto creation

one can restrict user creation for particular employee

Get email notification of login credentials for user

Configuration is available in Employees > Configuration > Settings

Access Rights are listed on Job Position, define access rights here in job position and it will linked to the relavent user of employee.







Go to settings > User & Companies > User and check access rights






In Employee user will find flag 'create user automatically?' so one can define user auto creation for particular employee. when flag true it will create user with login as work email of employee and password which is already defined in configuration and link that user in employee as related user in HR settings tab of employee.







By Enabling send password configuration user will get login credentials in mail notification.




By Enabling send link configuration user will get invitation link.






**This module is compatible for both the vesrion enterprise and community.
