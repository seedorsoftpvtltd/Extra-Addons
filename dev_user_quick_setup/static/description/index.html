Overview

In Odoo, To create user, then after create employee for the user, fill the nesesssary details in employee profile, then after create contract for the employee and assign leave allcatation to the employee. TO perform this all steps need to go different screens which take too much time and also chance to forget some steps so to overcome this problem here this Odoo application helps you to create New User with Contract, Employee and Leave Allocation on a single click on same screen.

Feature

Now you do not have to create Employee, Contract and Leave Allocation for New User separately go by one by one screen and menus
Create new user with his/her:
  • Create Employee with require fields details
  • Contract setup
  • Different Leave Allocations

How It Works

Create new User quickly from here

Enter details of Employee of new User
Enter details of Contract of new User
Enter details of Leave Allocation of new User
Send request to authenticated User

Authenticated user creates New User
