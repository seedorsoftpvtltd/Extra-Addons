Journal Entry Configurable Stages in Seedor

This app allows you to configure / create stages for your Customer Invoice / Vendor Bill / Credit Note / Refund / Journal Entries cycle and show it on form view.

Main Features:

  • Allow your Account manager to configure the stages for Customer Invoice / Bill / Credit Note / Refund.
  • Show Stages on Invoice / Bill / Credit Note / Refund / Journal Entries form of Seedor.
  • Note: These stages do not affect / change Seedor standard state field.
  • For more details please check below screenshots and watch the video.
