Construction Projects on Website Odoo App

This app allow you to view your construction projects on your website under Project.

Main Features:

  • Feature to create Construction Project Category and Set this category on Project form.
  • Allow you to add More Project Images for Galary on Website display.
  • Allow you to set Images/Photos to appear on Sliderbar of Project on Website.
  • Project Feature and Amenties for Project can be specifiy on Project form and display on Website.
  • Project Specification for Project can be specifiy on Project form and display on Website.
  • Contact Detail of Project can be fill on Project form and display on Website.
  • Project Floor Plan on for Project can be specifiy on Project form and display on Website.
  • Project Location Plan on for Project can be specifiy on Project form and display on Website.

Menus Available:

  • Project
    • Configuration
      • Project Categories
  • Website
    • Projects
