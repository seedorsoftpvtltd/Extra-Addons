TDS/TCS

This module helps to calculate TDS/TCS in purchase, sale and invoice.

Accounting

Step 1:

In Accounting -> Configuration -> Setting , TDS/TCS should be selected.

Step 2:

In Accounting -> Configuration -> Accounting -> TDS or TCS, Add New TDS / TCS .

Step 3:

In Accounting -> Sales -> Customers -> Sales & Purchases, TCS Applicalbe should be selected.

In Accounting -> Purchases -> Vendors -> Sales & Purchases, TDS Applicalbe should be selected.

Step 4:

In Accounting -> Sales -> Customer Invoices. Create Customer Invoice, If TCS Applicable for Selected Customer then TCS Calculate in Invoice.

In Accounting -> Purchases -> Vendor Bills. Create Vendor Bill, If TDS Applicable for Selected Vendor then TDS Calculate in Bill.
