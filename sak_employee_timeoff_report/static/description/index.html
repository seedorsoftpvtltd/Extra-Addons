Employee leave/Time-off report with all the required details
 in MS Excel format (XLSX)

REPORT CONTENT

  • Header Fields: From Date, To Date, Company Name.
  • Document Body:
        Fields: Employee, Position, Phone, Email, Department, Status, Create Date, Created by, Leave from, To, Leave Days, Description, Allocated Leave, Used Leave, Remaining Leave
  • Report Parameters: From Date, To Date
  • Configuration : There is not much configuration, just install the module and give security rights according to your needs in the menu (Timeoff Report), model (sak_employee_timeoff_report)
  • Menu Full Path: Timeoff / Reporting / Timeoff Report
  • Other Module Needed: "Report_xlsx"
