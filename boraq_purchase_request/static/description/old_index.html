Purchase Request

Odoo users can send a request for the management when they need to purchase anything which needs approval from Managers.

Configuration

1- In case the user doesn’t have access to purchase report

a. From user access role page

b. Navigate to “Purchase” module, then choose” Purchase Request”

c. In this case this type of users will have only ability to apply for a purchase request.



2- To Create a new purchase request, just navigate to “Purchase” Module



3- To create Purchase order:

a. Choose “Order Date” when you want to your order to be.

b. Choose the Approver name.

c. Inside “Description” fills it with items and quantities which you need to be purchased



d. Once the request is created, the approver will get notification, that he has new request waiting for his approvement



e. After Clicking on that notification, request details will appear, the details contain: request date, requester name, and description.



f. After clicking on “Approve” or “Reject” the requester will know what’s the status of his request.
