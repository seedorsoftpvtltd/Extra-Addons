Purchase Menu in Accounting

Overview

Allow Accountant to review confirmed purchase orders to allow to create vendor bill / refund

Features

  • No need to go to purchase module to generate invoice or choose po number to view products.

Configuration

No configuration needed

How to Work

From "Accounting -- Vendors -- Purchase Orders"

Notes:This module is tested and works on Odoo Community and Enterprise Edition.
