TELEMATEL | Disable Debug Mode

This module disables debug mode for users who do not have the "Enable debug mode" permission.

Features.
  • If any user activate Debug Mode without "Enable debug mode" permission automatically the user is logged out.
Set "Enable debug mode" permission to any user.

To enable user debug mode function, you have to configure user access rights in Settings.

  • Select Users or Manage Users to select user in user list.
      User Access Right Paths
    1. Setting / Users & Companies / Users
    2. Setting / Manage Users (Button)
  • Search and select user to set "Enable debug mode" permission.

By default no user has the permission activated, it must be done through the process shown
