Stock Picking Stages / Picking Stages

Delivery Order / Incoming Shipment Stages in Odoo

This app allows you to configure / create stages for your stock picking / shipments cycle and show it on transfer form.

Main Features:

  • Allow your inventory manager to configure the stages for transfers pickings/shipments.
  • Show Stages on Transfer / Picking form of Odoo.
  • Note: These stages do not affect / change Odoo standard state field.
  • For more details please check below screenshots and watch the video.
